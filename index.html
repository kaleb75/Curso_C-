Bienvenido al Curso de C++ Basico

C++ es un lenguaje de programación poderoso y versátil. Este curso está diseñado para principiantes que quieren aprender a programar desde cero.

  • ¿Qué aprenderás?
    • La sintaxis básica de C++.
    • Estructuras de control y funciones.
    • Introducción a la programación orientada a objetos (POO).
    • Manejo de archivos.
  • Requisitos:
    • Conocimientos básicos de informática.
    • Un computador con un compilador de C++ instalado (por ejemplo, Code::Blocks o Visual Studio Code).

Módulo 1: Estructura de un Programa en C++

Un programa en C++ básico tiene la siguiente estructura:

#include <iostream>
using namespace std;

int main() {
    // Tu código aquí
    return 0;
}

Explicación:

  • #include <iostream>: Librería estándar para entrada y salida.
  • using namespace std;: Usa el espacio de nombres estándar.
  • int main(): Punto de inicio del programa.

Módulo 2: Variables y Tipos de Datos

En C++, las variables se utilizan para almacenar datos. Algunos tipos comunes son:

Tipo Descripción Ejemplo
int Números enteros. int edad = 20;
float Números decimales. float altura = 1.75;
string Cadenas de texto. string nombre = "Juan";

Módulo 3: Operadores

Los operadores se utilizan para realizar operaciones matemáticas o lógicas:

  • Aritméticos: +, -, *, /, %
  • Relacionales: ==, !=, <, >
  • Lógicos: &&, ||, !

Ejemplo:

int a = 10, b = 5;
cout << (a + b); // Muestra 15
cout << (a > b); // Muestra 1 (true)

Módulo 4: Estructuras de Control

En C++, las estructuras de control permiten tomar decisiones y repetir acciones:

  • if: Para tomar decisiones.
  • for: Para bucles con un número fijo de iteraciones.
  • while: Para bucles que se ejecutan mientras una condición sea verdadera.

Ejemplo:

for (int i = 0; i < 5; i++) {
    cout << i << endl;
}

Módulo 5: Funciones

Las funciones son bloques de código que realizan una tarea específica y pueden ser reutilizados. La estructura básica de una función es:

      #include <iostream>
      using namespace std;
      
      // Declaración de la función
      int suma(int a, int b) {
          return a + b;
      }
      
      int main() {
          int resultado = suma(10, 20);
          cout << "La suma es: " << resultado << endl;
          return 0;
      }

Explicación:

  • int suma(int a, int b): Declara una función que toma dos parámetros de tipo entero y devuelve un entero.
  • return: Devuelve el resultado de la operación a + b.
  • int resultado = suma(10, 20): Llama a la función y almacena el resultado en una variable.

Ventajas de usar funciones:

  • Reutilización de código.
  • Mejor organización y legibilidad.
  • Fácil mantenimiento y depuración.

Módulo 6: Programación Orientada a Objetos (POO)

La POO es un paradigma de programación que organiza el código en **clases** y **objetos**. Una clase es como un molde, y un objeto es una instancia de esa clase.

Ejemplo básico:

      #include <iostream>
      using namespace std;
      
      // Definición de la clase
      class Persona {
      public:
          string nombre;
          int edad;
      
          // Método de la clase
          void saludar() {
              cout << "Hola, mi nombre es " << nombre << " y tengo " << edad << " años." << endl;
          }
      };
      
      int main() {
          // Crear un objeto de la clase Persona
          Persona persona1;
          persona1.nombre = "Juan";
          persona1.edad = 30;
      
          // Llamar al método
          persona1.saludar();
      
          return 0;
      }

Explicación:

  • class Persona: Declara una clase llamada Persona.
  • public: Especifica que los miembros de la clase son accesibles desde fuera de la clase.
  • persona1.nombre: Accede al atributo nombre del objeto persona1.
  • persona1.saludar(): Llama al método saludar.

Ventajas de la POO:

  • Organización del código en módulos (clases).
  • Reutilización mediante la herencia.
  • Facilidad para modelar problemas del mundo real.

Módulo 7: Manejo de Archivos

En C++, puedes trabajar con archivos para almacenar y recuperar datos. Usa las librerías fstream, ifstream (lectura) y ofstream (escritura).

Ejemplo básico de escritura y lectura:

      #include <iostream>
      #include <fstream>
      using namespace std;
      
      int main() {
          // Escritura en un archivo
          ofstream archivo("datos.txt");
          archivo << "Hola, este es un archivo de ejemplo." << endl;
          archivo.close();
      
          // Lectura desde un archivo
          ifstream archivoLectura("datos.txt");
          string linea;
          while (getline(archivoLectura, linea)) {
              cout << linea << endl;
          }
          archivoLectura.close();
      
          return 0;
      }

Explicación:

  • ofstream archivo("datos.txt"): Abre el archivo para escritura.
  • archivo.close(): Cierra el archivo después de escribir.
  • ifstream archivoLectura("datos.txt"): Abre el archivo para lectura.
  • getline(archivoLectura, linea): Lee cada línea del archivo y la almacena en la variable linea.

Consejos para trabajar con archivos:

  • Siempre cierra los archivos después de usarlos.
  • Verifica que el archivo se abrió correctamente antes de operar.
  • Utiliza rutas relativas para evitar problemas de localización.

Proyecto: Caja Registradora

Este proyecto simula una caja registradora básica en C++. Sus funcionalidades incluyen:

  • Saludar y despedirse del cliente.
  • Agregar productos al pedido.
  • Calcular el total de la cuenta.
  • Cobrar al cliente y devolver cambio.

Ejemplo de código:

      #include <iostream>
      #include <vector>
      #include <iomanip> // Para manejar formato decimal
      using namespace std;
      
      // Clase Producto
      class Producto {
      public:
          string nombre;
          double precio;
      
          Producto(string n, double p) : nombre(n), precio(p) {}
      };
      
      // Clase CajaRegistradora
      class CajaRegistradora {
      private:
          vector<Producto> productos;
          double total;
      
      public:
          CajaRegistradora() : total(0.0) {}
      
          void saludar() {
              cout << "=======================================" << endl;
              cout << "  Bienvenido a la Tienda La Esquinita!" << endl;
              cout << "=======================================" << endl;
          }
      
          void agregarProducto(string nombre, double precio) {
              productos.push_back(Producto(nombre, precio));
              total += precio;
              cout << "Producto agregado: " << nombre << " ($" << precio << ")" << endl;
          }
      
          void mostrarTotal() {
              cout << "=======================================" << endl;
              cout << "Total a pagar: $" << fixed << setprecision(2) << total << endl;
              cout << "=======================================" << endl;
          }
      
          void cobrar(double efectivo) {
              if (efectivo >= total) {
                  double cambio = efectivo - total;
                  cout << "Pago recibido: $" << efectivo << endl;
                  cout << "Cambio: $" << fixed << setprecision(2) << cambio << endl;
              } else {
                  cout << "Efectivo insuficiente. Faltan $" << fixed << setprecision(2) << (total - efectivo) << endl;
              }
          }
      
          void despedir() {
              cout << "=======================================" << endl;
              cout << "  Gracias por su compra. ¡Vuelva pronto!" << endl;
              cout << "=======================================" << endl;
          }
      };
      
      int main() {
          CajaRegistradora caja;
          caja.saludar();
      
          caja.agregarProducto("Manzana", 10.5);
          caja.agregarProducto("Pan", 15.0);
          caja.agregarProducto("Leche", 20.0);
      
          caja.mostrarTotal();
      
          double efectivo;
          cout << "Ingrese el efectivo entregado: ";
          cin >> efectivo;
      
          caja.cobrar(efectivo);
          caja.despedir();
      
          return 0;
      }

Explicación:

  • Producto: Clase para representar un producto con nombre y precio.
  • CajaRegistradora: Clase principal que contiene funcionalidades como saludar, agregar productos, mostrar el total, cobrar y despedirse.
  • vector: Contenedor dinámico para almacenar productos.
  • setprecision: Función de la librería iomanip para manejar el formato decimal.

Este proyecto muestra los conceptos clave de POO, como encapsulación, objetos y métodos. ¡Inténtalo!
